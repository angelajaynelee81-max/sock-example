# Pace Run Sock — Landing Page

A product landing page for Pace Performance socks.

Built with HTML, CSS and vanilla JavaScript.

## Features
- Product image gallery with 4 views
- Colour toggle (Midnight/Sage and Slate/Stone)
- Size selector with validation
- 360° image rotator
- Mobile responsive

## Built with
- HTML/CSS/JS
- Claude AI (code generation)
- Deployed on Vercel
